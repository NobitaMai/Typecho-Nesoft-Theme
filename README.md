# Typecho-Nesoft-Theme

#### 简介

基于Bootstrap的响应式主题！

项目主页：http://www.nesoft.cn

测试平台：typecho 1.0

==============================

#### 如何使用

点击右侧的`Download ZIP`按钮，下载完成之后解压得到plugins、resources、Nesoft三个文件夹，其中Nesoft为主题上传到'usr/themes'目录，resources文件夹放在usr目录，plugins文件夹下的插件放在'usr/plugins'目录，然后在后台启用主题。
